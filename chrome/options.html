PiQPull Settings

Organization ID

PiQPull auto-detects your Organization ID when a Claude.ai tab is open. You only need to enter it manually if auto-detect fails.

  1. Go to claude.ai/settings/account
  2. Copy your Organization ID — format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
  3. Paste below and click Save
Leave blank to rely on auto-detection (recommended).
